# 今回の目的　達成したい箇所 ユースケース

- ネットワーク環境の基本的なキーワード・ツールと構成の理解

## ネットワーク構成

- イーサネット（有線接続する際の通信規格）

  - WAN <-> ONU <-> NAT <-> LAN <-> ROOTER <-> HUB <-> DEVICE

- ワイヤレス

  - WAN <-> ONU <-> NAT <-> LAN <-> PoE <-> Access Point <-> DEVICE

## 各種キーワード・ツール

- WAN (ブロードバンド会社)

  - Wide Area Network（広域通信網）
    - 市街地から郊外、県外、海外に広がった広域のネットワーク
    - 点在する LAN（Local Area Network）と LAN を結ぶ公衆網
    - WAN を世界規模で実現しているのがインターネット
  - WAN では「グローバル IP アドレス」を使用する

- ONU

  - Optical Network Unit（光回線の終端装置）
    - ONU の役割は、光ファイバー(WAN)から送られる光信号をデジタル信号(LAN)に変換すること
    - 自宅に引き込んだ光回線から送られてくるのは光信号
    - 光信号の状態ではパソコンをインターネットに繋ぐことができない
    - ONU は契約している回線業者から貸し出される場合がほとんど
  - モデムも変換装置だが対象が異なる
    - モデムも ONU と同じ「変換装置」
    - ONU とモデムでは変換する信号が異なる
    - モデムの役割は電話回線などのアナログ信号をデジタル信号に変換すること
    - ADSL や CATV などのアナログ回線を使ってパソコンを使用する際には、ONU ではなくモデムが必要になってくる

- NAT(NAPT)

  - Network Address Translation（ネットワークアドレス変換）
    - プライベート IP アドレスとグローバル IP アドレスを双方向に変換する技術
    - 最近ではブロードバンドルーターや WAN ルーターが NAT の機能を備えている
    - NAT はグローバル IP アドレスを「隠す」という意味ではセキュリティの向上にもつながっている
  - NAPT（Network Address Port Translation）
    - IP アドレスだけでなくポート番号も変換する技術
    - 現在では NAPT を指して NAT と呼ぶのが一般的

- LAN

  - Local Area Network（ローカルエリアネットワーク）
    - 一定の限定されたエリアで接続できるネットワーク
  - LAN では「プライベート IP アドレス」を使用する

- PoE スイッチ

  - イーサネット（Ethernet）の通信ケーブルを利用して接続先の機器に給電する PoE
    - （Power over Ethernet）に対応したネットワークスイッチ（スイッチングハブ）
    - 電源を得るのが困難な場所に配置された機器を LAN ケーブルのみで駆動できる
  - トラブル発生時　-> 一番外側、要するにアクセスポイントから一番内側、PoE スイッチの順に切り分ける
    - アクセスポイントの故障 -> 新しいアクセスポイントに交換
    - 初期設定、PC の IP 設定(IPv)、取説 IP、取説の IP と同じセグメント、LAN＆PC、PING、check、browser、取説 IP で侵入、IP 設定（WAN || LAN)IP

- ルーター

  - 複数の端末を同時にインターネットに接続させるための装置
    - 現在はモデムや ONU とルーターの両方の機能を持つ一体型の装置がある
    - ルーター機能内蔵モデムやルーター機能内蔵 ONU があれば、複数台の端末を使うためにルーターを用意する必要はなくなる
  - 光回線が当たり前となった現在は、無線接続、つまり Wi-Fi を使う方が増えた
    - 有線も無線も使える Wi-Fi ルーター（無線 LAN ルーター）が主流
    - 最近では安定性と速度に優れた「光回線＋ IPoE」という通信方式が普及
    - ルーターを新しく用意するなら IPoE 対応のルーターを選ぶのがおすすめ

- ハブ（スイッチングハブ）
  - ルーターに複数台の端末を繋げることができる機器
    - ハブは単に LAN 配線を分岐して増やすだけの機器
    - コンセントの数が足りないときに使う OA タップと同義
